## When to Use

Use this skill when:
- Consuming broker APIs (REST)
- Integrating market data providers (WebSocket or REST)
- Configuring typed HTTP clients
- Implementing resilience policies (retry, circuit breaker)
- Propagating authentication headers between services

## Critical Patterns

### Pattern 1: Refit Interface

Define Refit interfaces in **Application/Interfaces**. The implementation is auto-generated by Refit.

```csharp
// Application/Interfaces/IExternalServiceClient.cs
public interface IExternalServiceClient
{
    [Get("/api/users/{id}")]
    Task<ExternalUserDto> GetUserAsync(string id);

    [Post("/api/users")]
    Task<ExternalUserDto> CreateUserAsync([Body] CreateExternalUserDto dto);

    [Get("/api/products")]
    Task<IEnumerable<ExternalProductDto>> GetProductsAsync(
        [Query] string? category = null,
        [Query] int? limit = null);
}
```

### Pattern 2: Refit Configuration with Polly

Configure Refit clients in **Infrastructure/DependencyInjection.cs** with resilience policies.

```csharp
// Infrastructure/DependencyInjection.cs
public static void ConfigureRefitClients(IServiceCollection services, IConfiguration configuration)
{
    // Register auth header handler
    services.AddTransient<AuthHeaderHandler>();

    // Retry policy with exponential backoff
    var retryPolicy = HttpPolicyExtensions
        .HandleTransientHttpError()
        .OrResult(msg => msg.StatusCode == HttpStatusCode.TooManyRequests)
        .WaitAndRetryAsync(
            retryCount: 3,
            sleepDurationProvider: retryAttempt =>
                TimeSpan.FromSeconds(Math.Pow(2, retryAttempt)),
            onRetry: (outcome, timespan, retryAttempt, context) =>
            {
                Console.WriteLine($"Retry {retryAttempt} after {timespan.TotalSeconds}s");
            });

    // Circuit breaker
    var circuitBreakerPolicy = HttpPolicyExtensions
        .HandleTransientHttpError()
        .CircuitBreakerAsync(
            handledEventsAllowedBeforeBreaking: 5,
            durationOfBreak: TimeSpan.FromSeconds(30));

    // Configure Refit client for broker/market data API
    var baseUrl = configuration["ExternalServices:BrokerApi:BaseUrl"]
        ?? throw new InvalidOperationException("BrokerApi:BaseUrl not configured");
    var timeout = configuration.GetValue<int>("ExternalServices:BrokerApi:Timeout", 30);

    services.AddRefitClient<IBrokerApiClient>()
        .ConfigureHttpClient(c =>
        {
            c.BaseAddress = new Uri(baseUrl);
            c.Timeout = TimeSpan.FromSeconds(timeout);
        })
        .AddHttpMessageHandler<AuthHeaderHandler>()
        .AddPolicyHandler(retryPolicy)
        .AddPolicyHandler(circuitBreakerPolicy);
}
```

### Pattern 3: AuthHeaderHandler

Handler to propagate API key or bearer token to external services.

```csharp
// Infrastructure/Services/AuthHeaderHandler.cs
public class AuthHeaderHandler : DelegatingHandler
{
    private readonly IConfiguration _configuration;

    public AuthHeaderHandler(IConfiguration configuration)
    {
        _configuration = configuration;
    }

    protected override async Task<HttpResponseMessage> SendAsync(
        HttpRequestMessage request,
        CancellationToken cancellationToken)
    {
        var apiKey = _configuration["ExternalServices:BrokerApi:ApiKey"];
        if (!string.IsNullOrEmpty(apiKey))
        {
            request.Headers.Add("X-API-Key", apiKey);
        }

        return await base.SendAsync(request, cancellationToken);
    }
}
```

### Pattern 5: Usage in Services

Inject the Refit client like any other dependency.

```csharp
// Application/Services/ExternalDataService.cs
public class ExternalDataService
{
    private readonly IExternalServiceClient _client;
    private readonly ILogger<ExternalDataService> _logger;

    public ExternalDataService(
        IExternalServiceClient client,
        ILogger<ExternalDataService> logger)
    {
        _client = client;
        _logger = logger;
    }

    public async Task<ExternalUserDto?> GetExternalUserAsync(string id)
    {
        try
        {
            var user = await _client.GetUserAsync(id);
            _logger.LogInformation("Retrieved external user {UserId}", id);
            return user;
        }
        catch (ApiException ex) when (ex.StatusCode == HttpStatusCode.NotFound)
        {
            _logger.LogWarning("External user {UserId} not found", id);
            return null;
        }
        catch (ApiException ex)
        {
            _logger.LogError(ex, "Error calling external API for user {UserId}", id);
            throw;
        }
    }
}
```

### Pattern 6: DTOs for External APIs

DTOs in `Application/DTOs/External/` to separate from internal DTOs.

```csharp
// Application/DTOs/External/ExternalUserDto.cs
public class ExternalUserDto
{
    [JsonPropertyName("id")]
    public string Id { get; set; } = string.Empty;

    [JsonPropertyName("email")]
    public string Email { get; set; } = string.Empty;

    [JsonPropertyName("full_name")]
    public string FullName { get; set; } = string.Empty;
}
```

## Resilience Policies (Polly)

### Retry Policy

| Parameter | Recommended Value | Description |
|-----------|-------------------|-------------|
| RetryCount | 3 | Number of retries |
| Backoff | Exponential | 2^attempt seconds |
| Jitter | Recommended | Avoids thundering herd |

### Circuit Breaker

| Parameter | Recommended Value | Description |
|-----------|-------------------|-------------|
| FailuresBeforeBreak | 5 | Failures before opening |
| DurationOfBreak | 30s | Time in open state |

## Decision Tree

```
How to consume external API?
  │
  ├── Is it a REST API (broker, market data)?
  │     └── Use Refit
  │
  ├── Needs retry?
  │     └── Add Polly WaitAndRetryAsync
  │
  ├── Is it a critical service that can fail?
  │     └── Add Circuit Breaker
  │
  └── Needs to propagate auth headers?
        └── Use AuthHeaderHandler
```

## Anti-Patterns (Guardrails)

| Anti-Pattern | Problem | Solution |
|--------------|---------|----------|
| **Direct HttpClient** | No typing, no resilience | Use Refit |
| **Infinite retry** | Can collapse service | Maximum 3-5 retries |
| **No circuit breaker** | Cascading failures | Add Polly |
| **No auth headers** | Unauthorized requests | Use AuthHeaderHandler |
| **No timeout** | Hanging requests | Configure timeout |
